# soil-erosion-narmada-basin-gee
This project focuses on estimating soil erosion in the Narmada Basin using the RUSLE (Revised Universal Soil Loss Equation) model in Google Earth Engine (GEE).  integrates  land use/land cover (LULC), rainfall, soil properties, and topography to compute key factors (R, K, LS, C, and P) and generate spatial soil loss maps.
